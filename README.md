🚀 Cool & Fun Bookmarklets
Welcome to the Cool & Fun Bookmarklets repo — a collection of tiny JavaScript tools you can run right from your bookmarks bar.
No installs, no extensions, just pure browser‑powered magic.

Bookmarklets let you run JavaScript on any webpage by saving code as a bookmark.
This repo gives you a bunch of copy‑paste‑ready snippets that do cool, weird, or useful things.

⭐ How to Use These Bookmarklets
Copy the code from any bookmarklet file in this repo.

Create a new bookmark in your browser.

Paste the code into the URL/location field.

Click the bookmark on any page to run it.

That’s it. Instant power.

📚 What’s Inside
Each bookmarklet lives in its own file and includes:

A short description of what it does

The JavaScript code you can paste into a bookmark

Notes or warnings if needed

Examples of bookmarklets you might find here:

🎉 Page confetti

🌈 Rainbow text mode

🔍 Element highlighter

🌀 Page spinner

🧼 Clear cookies for the current site (safe version)

🧪 Experimental visual effects

(Your actual list will depend on what you add.)

🧩 Adding Your Own Bookmarklets
Want to contribute or expand the collection?

Create a new .js file in the bookmarklets/ folder.

Include:

A title

A short description

The minified javascript: code

(Optional) a readable version of the code

Submit a pull request!

⚠️ Notes & Safety
Bookmarklets only run in your own browser, on pages you choose.

Some sites block scripts for security reasons — that’s normal.

Don’t run bookmarklets from strangers unless you trust the code.

This repo focuses on fun, harmless, and safe scripts.
